# Presentation du Sujet 

## Cadre du sujet
-   Multi-Agent Systems
-   Reinforcement Learning (RL)
-   Multi-Agent Reinforcement Learning (MARL)   
-   Jeux collaboratif/competitif

## Le But

Simplifier l'utilisation de techniques MARL/RL basées sur des librairies existantes pour faire jouer des agents à des jeux vidéos en fournissant des outils accessibles et nécéssitant peu de connaissances expertes. Il s'agit donc d'automatiser des tâches qui sont habituellement effectuées manuellement tel que :

- Pré-configuration : la prise en compte d'un environnement.
- Configuration :  la prise en compte d'algo MARL/RL avec une recherche d'oprtimisation des hyper-paramètre (HPO).
- Entrainement : entrainer les agents sur un environnement donné.
- Test : confronter les agents à de nouveaux environnements.

## Démonstrateur:
- Construire un Framework Python qui fournit une API (Library) permettant à un utilisateur peu familier du RL/MARL de :
	- Choisir un ensemble d'environnements collaboratifs.
	- Choisir une méthode de résolution (un algorithme MARL/RL) ou laisser un processus de résolution automatique s'opérer.
- S'appuyer sur l'API pour fournir une interface CLI ou GUI (React/Angular/VueJS) afin d'améliorer encore davantage l'utilisabilité
